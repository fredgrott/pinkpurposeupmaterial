SetUp

Basic SetUp

This web template is self contained with all the icons, images, fonts, css, and js to get started with your web project straight-away. You will start by replacing favicons, images and content with your own.


As you change from the flat-file structure of the web template you will go about changing the menu url hrefs. An easy way to do that is start with the sitemap and pages fiels as they list most of the pages so you can use them affter you modify them to flight check if you go the modified url hrefs right, etc.

Advance SetUp

You can change the loading of several CSS, JS, Fonts to loading from the cloud. The specific ursl you can find below to modify the head of each file to load stuff from the cdn cloud.

CSS HTML Arcbhitecture

I set up my CSS and HTML for the most part to avoid sticking helper classes for positioning, color, etc in my HTML so that for the most part you can customize strictly with the custom.css file. This way its easy for you to theme a new project based on this web template in a fast and easy manner.

CDN URLs

The CDN urls to use to load resoruce dependencies from the cloud:

Roboto Font
<link href="https://fonts.googleapis.com/css?family=Roboto:100,100i,300,300i,400,400i,500,500i,700,700i,900,900i" rel="stylesheet">

<link href="https://fonts.googleapis.com/css?family=Roboto+Slab:100,300,400,700&subset=cyrillic,cyrillic-ext,greek,greek-ext,latin-ext,vietnamese" rel="stylesheet">
